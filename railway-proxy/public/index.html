How do I know this isn't rigged?
Normally you'd just have to trust the casino didn't stack the deck. Here you don't have to — you can check.

This game runs inside a sealed box called an enclave. Once it's sealed and running, nobody — not even the people who built it — can quietly swap in different code without it being obvious.

Here's the trick: the sealed box has a fingerprint — one giant number that's tied to the exact program running inside. Change even one letter of code and the fingerprint comes out completely different.

Every single time your browser talks to the game, it checks that fingerprint against the one it already expects (pinned to the exact code posted publicly). If they don't match, your browser refuses to send or receive anything — you'd see a big red warning screen instead of cards. That's the badge at the top of this page, doing that check on every move you make.

So: you (or anyone) can read the actual source code, check the fingerprint yourself, and know for certain the shuffle and rules are exactly what's published — not a secretly-swapped version.
What this doesn't prove: that any one shuffle came out fair by chance. It proves the shuffle code wasn't tampered with — not that every individual card order is independently provable after the fact. That would need an extra step (called "commit-reveal") that isn't built here.
